# First Application

Projeto de landing page pessoal desenvolvido por outra pessoa. Estou contribuindo com melhorias de layout, responsividade e refinamento visual.

## Sobre o projeto
Este site apresenta uma desenvolvedora e reúne seções de introdução, projetos, skills e contato. O objetivo é oferecer uma presença online simples, elegante e fácil de expandir.

## Minha contribuição
- Modernização do layout com tipografia e paleta mais atuais.
- Ajustes de responsividade para diferentes tamanhos de tela.
- Organização visual por seções e cartões para leitura mais fluida.
- Pequenas melhorias na hierarquia de conteúdo.

## Tecnologias
- HTML5
- CSS3

## Como visualizar
Abra o arquivo `index.html` no navegador.
